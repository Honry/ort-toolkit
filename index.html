ONNX Runtime Toolkit

Usage

Some parameters are supported in url, and you may use them as 'index.html?key0=value0&key1=value1...'. Supported parameters are:

1) warmupTimes: Warmup times

2) runTimes: Run times

3) tasks=[task0,task1,task2]: tasks to run, split by ','. Candidates are 'conformance', 'performance', 'ortProfiling', 'webgpuProfiling'.

4) ortUrl: ort url. Example: ortUrl=https://wp-27.sh.intel.com/workspace/project/onnxruntime

5) modelLocation=[github|local]. Local models are in onnx-models folder, while github ones are at https://github.com/webatintel/onnx-models.
